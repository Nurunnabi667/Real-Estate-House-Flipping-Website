# Real Estate House Flipping Website

This is a WordPress website project for a New Jersey-based real estate investment and house flipping business.

## Project Overview

- **Home Page:** Landing page highlighting business, services, and CTAs
- **Wholesaler Page:** Lead submission form for off-market properties
- **JV Partnership Page:** Lead form and collaboration info for partners
- **Construction Consulting:** Consulting services with tailored quotes
- **About Us:** Company introduction

## Tech Stack

- WordPress
- PHP, HTML, CSS
- Elementor / Gutenberg (for page building)
- WPForms / Contact Form 7
- SEO: Yoast SEO / RankMath
